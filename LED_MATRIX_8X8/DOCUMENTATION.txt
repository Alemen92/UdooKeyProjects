#-----------------ENG----------------------#

Folder Description: The folder contains the command files for an 8x8 LED matrix module (MAX7219). There are two subfolders: one named LIB, where the programs use the MicroPython library associated with the module (micropython-max7219), which can be easily installed via Thonny, and another named NO_LIB, which configures the module without using the library. Each folder will have various versions that expand the possible functions, making it suitable for integration into larger projects.

Materials: Python sketches and various documentation.

Requirements for Testing: UDOO KEY, Raspberry Pi Pico, or other boards compatible with MicroPython.
