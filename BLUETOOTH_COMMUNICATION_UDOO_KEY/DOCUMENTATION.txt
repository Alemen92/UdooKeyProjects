#-----------------ENG----------------------#

Objective: Control the ESP32 module of UDOO Key via BLE using a GUI to turn the two LEDs on and off on the board.

Materials: UDOO Key or ESP32 module with two LEDs to connect.

Files: bluetooth_communication_interface.py, boot_bluetooth_pico.py, and BLE_scanner.py

Notes: The connection is made via MAC address, so it is recommended to use the scanner program to identify it.
