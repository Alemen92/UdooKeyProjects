#-----------------ENG----------------------#

Goal: Establish communication between two ESP32 modules, where one functions as an Access Point and the other as a station that receives the signal and turns on a LED (one of the default LEDs on the controller).

Materials: Two UDOO KEY boards. Alternatively, ESP32 controllers can be used since the Raspberry Pi Pico module on the board is not required.

Files: For all versions directories AP_boot.py for the access point, STA_boot.py for the station, PC_STA.py to use your PC, if equipped with Wi-Fi, as a station.
